## March 13, 2023

- Multimedia
  - `video` tag
    - `width`
    - `height`
    - `autoplay`
    - `muted`
    - `controls`
    - can have text inside the video tag
    - has a nested, self-closing `source` tag
      - has `src="video.mp4"` and `type="video/mp4"`
  - `audio` tag
    - has the same nested source and type
    - has same autoplay/muted/controls
